# Aly Morsy — Personal Research Portfolio

Static personal portfolio prepared for GitHub Pages.

## Public address

After publishing this repository under the name:

`alinassertum00.github.io`

the website will be available at:

`https://alinassertum00.github.io`

## Publishing

1. Create a new public GitHub repository named exactly `alinassertum00.github.io`.
2. Upload all files in this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then save.

The site normally becomes available within a few minutes.

## Optional custom domain

To use a domain such as `www.alymorsy.com`, buy the domain and add it in:

**Settings → Pages → Custom domain**

Do not add a `CNAME` file until the domain has been purchased and confirmed.
